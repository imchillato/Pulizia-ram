# Pulizia-ram
Questo script in CSharp permette di utilizzare SetProcessWorkingSetSize, un comando che fa parte di una libreria chiamata: kernel32.dll, sostanzialmente il codice usa una libreria per chiudere un processo in modo forzato. e dopo la pulizia si usa il Garbage Collector per pulire la ram che il cleaner ha occupato per liberare la ram in eccesso.
